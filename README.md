# Experimental settings and datasets

This repository contains information aimed at reproducing the experiments on multidomain machine translation described in two published papers:
- Minh Quang Pham, Josep Crego, François Yvon, Jean Senellart. Revisiting Multi-Domain Machine Translation, to appear in Transactions of the ACL, 2020
- Minh Quang Pham, Josep Crego, François Yvon, Jean Senellart. A Study of Residual Adapters for Multi-Domain Neural Machine Translation, to appear in the proceedings of the conference on Machine Translation (WMT) 2020
